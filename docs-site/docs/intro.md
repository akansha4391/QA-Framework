# Introduction

Welcome to the **Enterprise QA Automation Framework** documentation!

This framework is designed to provide a unified, scalable, and robust solution for all your testing needs, including UI, API, and Performance testing.

## Key Features

- **Hybrid UI Driver**: Seamlessly switch between Selenium and Playwright.
- **Unified API Client**: Simple yet powerful wrapper for REST and GraphQL.
- **Data Management**: Built-in support for CSV, YAML, JSON, and Databases.
- **Performance**: Integrated Locust runners.
- **Reporting**: Native Allure integration.
